# Visor Multimedia

> ⚠️ **AVISO**: Este proyecto es un **trabajo escolar** y no debe ser tratado como software profesional o listo para producción. Puede contener errores, malas prácticas y código poco optimizado. Úsalo bajo tu propia responsabilidad.

## Descripción

Reproductor multimedia simple desarrollado en C# con Windows Forms. Permite reproducir archivos de audio y video, con controles básicos de reproducción como play, pausa, detener y ajuste de volumen.
